聊天室发送图片功能

前几天在做毕业设计，做了一个基于Node的聊天室，但是此聊天室只能发送文字，当然不能满足用户最基本的需求，所以想加一下发送图片的功能。
那么发送图片该怎么来做，怎么把自己本地的图片发送给聊天室内的其他人？
我总结了以下步骤：

  • 通过input(file)选取本地的图片
  • 通过Ajax把图片传给后台接口
  • 后台接收到图片后将其上传到又拍云帮助我们来存储
  • 将存储到又拍云上的图片再下载到本地的临时文件里
  • 再把图片通过Socket.IO发送出去
  • 这样，整个过程就完成了
